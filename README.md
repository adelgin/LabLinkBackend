# Проект LabLink

backend для соцсети для научных сотрудников с интеграцией с научными сервисами индексации

Стек
- Java
- Spring Boot
- postgres

Проект реализует логику подтягивания данных из научных API при регистрации, что обеспечивает автозаполнение профиля пользователя
В проекте есть логика для работы пользователей, постов, лайков и комментариев, ленты с весовой системой рекомендаций.
Настроено взаимодействие с проектом LabLinkIntegration 

# Разворот проекта

Чтобы развернуть проект применяем команду
```bash
docker-compose up --build
```
Из корневой папки проекта
